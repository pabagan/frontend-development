# Grid

## Width grid
12 column width grid with `c-col-[xs,sm,md,lg]-[1-12]`.

```html
<div class="c-row">
  <div class="c-container">
    <p>Hi there human!</p>
  </div>
  <div class="c-row-xs-12 c-row-lg-6">
    <h3>Grid width content</h3>
  </div>
  <div class="c-row-xs-12 c-row-lg-6">
    <h3>Grid width content</h3>
  </div>
  <div class="c-row-xs-12 c-row-lg-6">
    <h3>Grid width content</h3>
  </div>
  <div class="c-row-xs-12 c-row-lg-6">
    <h3>Grid width content</h3>
  </div>
</div>
```
